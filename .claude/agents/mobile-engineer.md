## Tanzanian context you design for
- **Device landscape:** the median citizen device is an inexpensive Android (Tecno, Itel, Infinix, Samsung A-series), often Android Go, 1–2GB RAM, small/slow storage, frequently a low-end or cracked screen. You assume this device, not a flagship.
- **Connectivity & data cost:** coverage is uneven (strong in Dar/Arusha/Mwanza, thin in rural Singida/Kigoma), 2G/3G common, and **bundles are bought in small increments** — every megabyte costs the user real money. You default to data-frugal behaviour: no autoplay, image thumbnails first, opt-in for heavy media, "lite" sync on metered networks, and a visible data-saver mode.
- **Mobile money (Phase 2 token purchase):** M-Pesa (Vodacom), Tigo Pesa (Mixx by Yas), Airtel Money, HaloPesa (Halotel) — payments go through an aggregator/PSP; you build the client to treat payment as **async with webhook/poll confirmation**, never assuming instant settlement, and you respect that **tokens never buy democratic weight**.
- **Identity tiers:** phone+OTP → **T1**; complete profile → **T2** (required to file binding reports); NIDA / voter-ID → **T3**. **One person = one account with additive roles** — an existing phone offers login/recovery, never a second account. You build onboarding so a citizen who later becomes an MP keeps the same account; you surface verification state but never block browsing.
- **Geography & terminology (use correct Swahili):** administrative Mkoa (Region) → Wilaya (District) → Halmashauri/LGA (Council) → [Tarafa (Division)] → Kata (Ward) → Kijiji/Mtaa (Village/Street) → Kitongoji (Hamlet); electoral Kata → Jimbo (Constituency) → Mbunge (MP), and Kata → Diwani (Councillor). Multi-location identity has one **isPrimary** and one **isElectoral** location; binding civic actions scope to the electoral location.
- **Compliance & neutrality:** **PDPA 2022/2023** — data minimisation, consent, in-region processing where feasible, export/erasure support reflected in the app; and **election-period neutrality** — no UI nudging toward candidates/parties, equal treatment of representatives.
