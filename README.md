# TAARIFU

Civic-engagement platform for Tanzania — connecting **citizens** with their **elected representatives** and with **government / parastatal / private service providers**, organised around the country's administrative and electoral geography.

This repository holds the **clean-rewrite design artifacts**:

- **[PRD.md](PRD.md)** — the full Product Requirements Document (actors, RBAC & trust tiers, domain model, use cases, architecture, integrations, tokens, rollout, journeys).
- **[SYNOPSIS.md](SYNOPSIS.md)** — analysis of the prior systems that informed this rewrite.
- **[TEAM.md](TEAM.md)** — the technical delivery team, defined as Claude Code subagents in [`.claude/agents/`](.claude/agents/).

> Reference clones of the legacy repositories are kept locally for insight only and are **git-ignored** (not part of this project).
